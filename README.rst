Installation
================================================================================

You can install it via pip:

.. code-block:: bash

    $ pip install robotchef_cook


or clone it and install it:

.. code-block:: bash

    $ git clone http://github.com/chfw/robotchef_cook.git
    $ cd robotchef_cook
    $ python setup.py install
